Setting up Anki

Before importing your CSV, you need to create a note type in Anki with the right fields and card templates. This only needs to be done once.

Create the note type

1
Open Anki and go to Tools > Manage Note Types
2
Click Add, choose Add: Basic, and name it Kindle Spanish Vocab
3
Select your new note type and click Fields. Delete the default "Front" and "Back" fields, then add these seven fields in order:
word
definition
label
book
authors
usage
morphology
4
Click Save to close the Fields window, then click Cards to open the template editor.
5
Replace everything in the Front Template with the code below:

Importing your CSV

1
In Anki, go to File > Import and select your downloaded CSV file
2
Set the Note Type to Kindle Spanish Vocab
3
Make sure Allow HTML in fields is checked (this preserves the word highlighting in your context sentences)
4
Verify the field mapping matches: field 1 = word, field 2 = definition, field 3 = label, field 4 = book, field 5 = authors, field 6 = usage, field 7 = morphology. Ignore any extra columns (morphology_source, status, fail_reasons).
5
Click Import. Your cards are ready to study.
